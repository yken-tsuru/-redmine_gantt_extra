# Redmine Gantt Extra

標準のガントチャートの使い勝手を向上させ、新機能を追加するRedmineプラグインです。

## 機能

### 1. ドラッグ＆ドロップによるスケジュール変更
ガントチャート上のタスクバーをドラッグ＆ドロップすることで、開始日と期日を直接変更できます。
- タスクを左右にドラッグして日付を移動できます。
- 更新はAJAX経由で即座に保存されます。
- 権限設定に従い、編集権限を持つユーザーのみが操作可能です。

### 2. ツリー表示フィルタ
特定のタスク階層にフォーカスした「ツリー表示」モードを提供します。
- ガントチャートのオプションに「親チケットID（Parent Issue ID）」フィルタを追加します。
- 親チケットIDを指定すると、そのチケットとすべての子孫チケット（子、孫など）のみをツリー状に表示します。
- 関係のないチケットは非表示になり、見通しが良くなります。

## インストール

1.  Redmineの `plugins` ディレクトリにリポジトリをクローンします:
    ```bash
    cd /path/to/redmine/plugins
    git clone https://github.com/your-repo/redmine_gantt_extra.git
    ```
2.  Redmineを再起動します。

## 使い方

### ドラッグ＆ドロップ
ガントチャート画面（`/projects/xxx/issues/gantt`）を開くだけです。編集権限があれば、表示されているタスクバーをドラッグできます。

### ツリー表示
1.  ガントチャート画面を開きます。
2.  フィルタエリアにある「親チケットID」入力欄を探します。
3.  表示したいルートチケットのIDを入力します。
4.  「適用」をクリックします。
5.  指定したチケットをルートとするツリー構造が表示されます。
