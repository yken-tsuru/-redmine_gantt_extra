# Redmine Gantt Extra

標準のガントチャートの使い勝手を向上させ、新機能とプレミアムなUIを追加するRedmineプラグインです。

## 主な機能

### 1. 直感的な操作 (Drag & Drop / Resize)
- **日程移動**: ガントチャート上のタスクバーをドラッグして、開始日と期日をセットで変更。
- **期間変更 (リサイズ)**: タスクバーの端をドラッグして、開始日または期日を個別に調整。
- **親子関係の変更**: チケット名をドラッグ＆ドロップして、親チケットを直感的に付け替え。
- **リアルタイム更新**: 変更はAJAX経由で即座に保存されます。

### 2. 日付表示モード (Compact Mode)
- **表示切替**: 右上の「日付表示に切替」ボタンで、週番号の代わりに日付を直接表示。
- **省スペース**: ヘッダーをスリム化し、チャートエリアを広く活用。
- **状態維持**: ブラウザを更新しても、選択した表示モードが維持されます。

### 3. クイック編集ポップアップ
- **即時編集**: ガントバーをクリックするだけで編集用ポップアップを表示。
- **編集項目**: 開始日、期日、進捗率、担当者をその場で変更可能。
- **スムーズな体験**: ページ遷移なしで素早く情報を更新できます。

### 4. 高度なツリー表示フィルタ
- **階層フォーカス**: フィルタに「親チケットID」入力欄を追加。
- **子孫表示**: 指定した親チケットとその配下のすべての子孫チケットのみを表示し、特定のプロジェクトツリーに集中できます。

### 5. プレミアム・デザイン
- **モダンUI**: グラスモーフィズム（背景ぼかし）と滑らかなアニメーションを採用。
- **視覚的フィードバック**: リサイズハンドルのグリップ表示や、週末のハイライト表示により、操作ミスを軽減。
- **完全な多言語対応**: すべてのUIラベルが国際化されており、日本語と英語に完全対応しています。

## インストール

1.  Redmineの `plugins` ディレクトリにリポジトリをクローンします:
    ```bash
    cd /path/to/redmine/plugins
    git clone https://github.com/yken-tsuru/redmine_gantt_extra.git
    ```
2.  Redmineを再起動します。

## 使い方

### 日付表示の切り替え
ガントチャート右上のコンテキストメニューにある「日付表示に切替」をクリックすると、表示がコンパクトな日付ベースに変わります。

### クイック編集
表示されているタスクバーをクリックするとポップアップが開きます。内容を変更して「保存」をクリックしてください。

### 親チケットの付け替え
左側のチケット一覧のチケット名をドラッグし、別のチケット名の上にドロップすると、親チケット変更の確認ダイアログが表示されます。
